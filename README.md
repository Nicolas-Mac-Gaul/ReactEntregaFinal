# 🎮 GameStore - Ecommerce Gamer

## 📌 Descripción

GameStore es una aplicación web de comercio electrónico orientada a la venta de productos gamer y tecnología.

El proyecto permite visualizar productos, consultar detalles, agregarlos al carrito, gestionar usuarios mediante autenticación y administrar el catálogo utilizando Firebase Firestore.

Fue desarrollado con React aplicando componentes reutilizables, Context API, React Router y herramientas modernas de diseño.

---

# 🚀 Tecnologías utilizadas

- React + Vite
- JavaScript
- React Router DOM
- Firebase Authentication
- Firebase Firestore
- Context API
- React Bootstrap
- Styled Components
- React Icons
- React Helmet Async
- CSS3

---

#  Funcionalidades principales

## 🛒 Carrito de compras

- Agregar productos al carrito.
- Modificar cantidades.
- Eliminar productos.
- Vaciar carrito.
- Manejo global mediante Context API.

---

## 🔐 Autenticación de usuarios

- Registro de usuarios.
- Inicio de sesión.
- Cierre de sesión.
- Persistencia mediante Firebase Authentication.
- Protección de rutas privadas.

Las secciones administrativas requieren un usuario autenticado.

---

## 📦 Gestión de productos (CRUD)

Panel de administración:

- Crear productos.
- Leer productos desde Firestore.
- Editar productos existentes.
- Eliminar productos.
- Validaciones de formularios.
- Confirmación antes de eliminar.

---

## 🔎 Búsqueda de productos

Incluye búsqueda dinámica:

- Filtrado en tiempo real.
- Actualización automática mientras el usuario escribe.

---

## 📱 Diseño responsive

La aplicación utiliza:

- React Bootstrap.
- Sistema de grillas responsive.
- Componentes adaptables para celulares, tablets y escritorio.

---

## ⚡ Optimización y experiencia de usuario

Incluye:

- Spinners durante cargas.
- Mensajes de error.
- Íconos visuales con React Icons.
- SEO dinámico con React Helmet.
